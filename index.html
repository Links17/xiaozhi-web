Xiaozhi - iOS Styled Web Application

A beautifully designed iOS-inspired web application built with Next.js, Tailwind CSS, and shadcn/ui.

Features

iOS Design System

Clean, minimal, and elegant design following iOS design guidelines with attention to detail in typography, spacing, and color schemes.

Authentication

Complete user authentication flow with login, registration, and password reset functionality, all styled with iOS aesthetics.

Task Management

Comprehensive task management interface with the ability to create, edit, and track tasks with different priority levels.

Responsive Design

Fully responsive design that works beautifully on all devices, from mobile phones to desktop computers.

Project Description

Xiaozhi is a modern web application that showcases the elegant simplicity of iOS design applied to web interfaces. Built with Next.js for performance and Tailwind CSS for styling, the application features a complete authentication system and a task management interface that adheres to iOS design guidelines.

The project emphasizes clean typography, appropriate use of white space, subtle shadows, and an iOS-consistent color palette to create a familiar and intuitive user experience for iOS users on the web.
